개인정보처리방침 (장보기 냉장노트)

시행일: 2026-02-19

장보기 냉장노트(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 개인정보처리방침은 앱에서 어떤 정보를 어떻게 처리하는지 설명합니다.

1. 수집하는 정보

  • 이용자가 직접 입력한 정보: 장보기/냉장고 항목, 메모, 수량, 유통기한, 보관 분류
  • 영수증 스캔 정보: 촬영/선택한 이미지, OCR 텍스트 인식 결과
  • 알림 정보: 유통기한 알림 예약/표시를 위한 로컬 알림 데이터

2. 정보 이용 목적

  • 장보기 리스트 및 냉장고 재고 관리 기능 제공
  • 유통기한 임박 알림 제공
  • 영수증 OCR을 통한 품목 자동 추출 기능 제공

3. 권한 사용 안내

  • 카메라 권한: 영수증 촬영
  • 사진(갤러리) 접근 권한: 영수증 이미지 선택
  • 알림 권한: 유통기한 알림 제공

4. 보관 위치 및 기간

  • 앱 데이터는 기본적으로 이용자 기기 로컬 DB에 저장됩니다.
  • 앱 개발자가 운영하는 별도 서버로 전송/저장하지 않습니다.
  • 영수증 처리 이미지의 임시 파일이 기기 내부 저장소에 생성될 수 있습니다.
  • 이용자는 앱 내 삭제 기능 또는 앱 삭제를 통해 데이터 제거가 가능합니다.

5. 제3자 제공 및 국외 이전

  • 개발자는 이용자 정보를 제3자에게 판매/제공하지 않습니다.
  • 앱이 직접 운영하는 외부 서버로 개인정보를 전송하지 않습니다.
  • OS/플랫폼 및 라이브러리의 동작 특성에 따른 필수 처리만 발생할 수 있습니다.

6. 이용자 권리

  • 앱 내 데이터 열람/수정/삭제
  • 알림 권한 해제
  • 앱 삭제를 통한 데이터 제거
