# 개인정보처리방침 (장보기 냉장노트)

시행일: 2026-02-19

장보기 냉장노트(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 개인정보처리방침은 앱에서 어떤 정보를 어떻게 처리하는지 설명합니다.

## 1. 수집하는 정보
앱은 회원가입 기능이 없으며, 다음 정보를 기기 내에서 처리합니다.

- 이용자가 직접 입력한 정보
  - 장보기 항목, 냉장고 보관 항목, 메모, 수량, 유통기한, 분류(냉장/냉동/상온) 등
- 영수증 스캔 관련 정보
  - 카메라 촬영 이미지 또는 갤러리에서 선택한 이미지
  - 이미지 내 텍스트 인식(OCR) 결과
- 알림 정보
  - 유통기한 알림 예약/표시를 위한 로컬 알림 데이터

## 2. 정보 이용 목적
앱은 수집·처리한 정보를 다음 목적에만 이용합니다.

- 장보기 리스트 및 냉장고 재고 관리 기능 제공
- 유통기한 임박 알림 제공
- 영수증 OCR을 통한 품목 자동 추출 기능 제공

## 3. 권한 사용 안내
앱은 기능 제공을 위해 아래 권한을 사용할 수 있습니다.

- 카메라 권한: 영수증 촬영
- 사진(갤러리) 접근 권한: 영수증 이미지 선택
- 알림 권한: 유통기한 알림 제공

이용자는 기기 설정에서 권한을 허용/거부할 수 있습니다.

## 4. 보관 위치 및 기간
- 앱 데이터는 기본적으로 이용자 기기 내 로컬 DB(예: Isar)에 저장됩니다.
- 앱 개발자가 운영하는 별도 서버로 전송/저장하지 않습니다.
- 영수증 처리 이미지의 임시 파일은 앱 처리 과정에서 기기 내부 저장소에 생성될 수 있습니다.
- 이용자는 앱 내 삭제 기능 또는 앱 삭제를 통해 데이터 제거가 가능합니다.

## 5. 제3자 제공 및 국외 이전
- 앱 개발자는 이용자 정보를 제3자에게 판매/제공하지 않습니다.
- 앱이 직접 운영하는 외부 서버로 개인정보를 전송하지 않습니다.
- 단, OS/플랫폼(예: Android/iOS) 또는 사용 라이브러리의 동작 특성에 따라, 서비스 제공에 필요한 범위 내에서 기기 환경 정보가 처리될 수 있습니다.

## 6. 아동의 개인정보
앱은 아동을 포함한 모든 이용자가 사용할 수 있으나, 별도의 회원가입/프로필 수집을 하지 않습니다. 보호자는 아동의 기기 권한 및 앱 사용을 관리해 주세요.

## 7. 이용자 권리
이용자는 언제든지 다음을 수행할 수 있습니다.

- 앱 내 데이터 열람/수정/삭제
- 알림 권한 해제
- 앱 삭제를 통한 데이터 제거

## 8. 보안
앱은 로컬 저장 중심으로 설계되어 있으며, 불필요한 데이터 전송을 최소화합니다. 다만 이용자 기기 자체의 보안(잠금화면, OS 업데이트 등)은 이용자 책임입니다.

## 9. 개인정보처리방침 변경
법령 또는 기능 변경 시 본 방침은 업데이트될 수 있으며, 변경 시 저장소 문서를 통해 공지합니다.
